## Project Overview
---
The aim of this project is to analyze and clean messy data related to patients' health records. The goal is to help enhance critical thinking, attention to detail, and problem-solving skills in data cleaning. 

## Data Source
---
The main source is the GreenData Dirty Healthcare Data csv
